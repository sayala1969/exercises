Adjustable box

In this exercise, you will learn how to manipulate the DIV element by adjusting the width and height of a box using sliders.

Tasks

  • Please don't open "dont-look-dom-adjustments.js" file until you are done with the exercise.
  • You can un-comment line #7 in this file to see the final result.
  • Read this html file and inspect the structure of each element to learn about different types of tags and attributes.
  • Go through the css file to learn about different styles used to make up this page.
  • use "dom-adjustments-solution.js" file to write your code.
  • [Easy] Use JavaScript to listen for changes on the sliders and adjust the width and height of the box accordingly
  • [Medium] Display the current dimensions of the box inside the box itself (e.g., "100x100")
  • [Hard] Ensure that the dimensions update in real-time as the sliders are adjusted
