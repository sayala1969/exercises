Catch me if you can

In this exercise, you will learn how to manipulate the DOM by making a button run away from the mouse cursor.

  • Use JavaScript to listen for mouse hovering (mouseenter event) on the button and adjust its position accordingly.
  • Ensure that the button moves in a random direction away from the cursor each time it is hovered over.
  • Make sure the button stays within the bounds of the viewport.
  • Bonus: Add an event to "Freeze" button that, when clicked, stops the "Catch me if you can!" button from moving.
  • Add a click event to "Catch me if you can button" that, when clicked, it will display the scrim DIV element.
